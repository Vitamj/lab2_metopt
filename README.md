# Вторая лабораторная работа

Глобальная задача реализовать методы учитывающие что функция может быть НЕ непрерывная (реализовать стахостические методы оптимизации 2 штуки). И протестировать их на определенных [функциях](https://ru.wikipedia.org/wiki/Тестовые_функции_для_оптимизации) и  функцию из [десмоса](https://www.desmos.com/3d/cfh7o2ckkx).

## Требование

Необходимо разобраться в стахостических методах оптимизации и реализовать их поверх КОНСТРУКТИВНОГО числа.

Необходимо провести сравнение скорости сходимости, количества вызовов функции и требуемой памяти для реализованных методов в этой лабораторной работе и в предыдущей. Сравнение ожидается табличным и по возможности графическим.

Требуется теоретическое пояснение к тому почему конкретный метод уступает/превосходит иные методы.
